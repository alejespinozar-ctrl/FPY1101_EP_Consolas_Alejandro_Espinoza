# Sistema de Gestión de Consolas de Videojuegos

## Descripción:
El sistema sirve para poder ver cuantas consolas hay en stock, las marcas, modelos y precio de las consolas. el sistema sirve mucho para la gente que tiene una tienda que vende consolas ya que se puede ver el stock que hay y el precio a cual se vende.

## Funcionalidades:
-El sistema puede agregar consolas
-El sistema puede buscar la consola que el usuario agrego y la puede buscar con el nombre que le puso en la sigla
-El sistema puede eliminar las consolas que el usuario agrego, eso sirve en caso de que el usuario se allá quedado sin stock o desee elimnarla
-El sistema puede mostrar todas las consolas que estan agregadas, con sus respectivos stock de cada consola, su modelo y el precio

## Cómo ejecutar:
Primero: tienes que darle al boton de play en la parte superior derecha para poder correr el codigo
Segundo: tienes que seleccionar alguna opcion valida del 1 al 5, si no ingresas una opcion valida el programa de va a decir "Opción inválida. Ingrese un número del 1 al 5."
Tercero: Tienes que agregar el nombre sigla ej "XSX = (Xbox Series X)", despues tienes que agregarle el nombre de la consola ej "Xbox Series X", despues tienes que agregar el nombre del fabricante ej "Microsoft", despues tienes que ingresar el precio y el stock que tienes o deseas agregar
Cuarto: Si ingresas a la opcion 2 despues de ingresar todo correctamente en la primera opcion, te debe pedir el nombre con el cual lo agregaste "sigla" y te va aparecer todos los datos que ingresaste de la consola que buscaste 
Quinto: Si ingresaste a la tercera opcion puedes eliminar la consola que agregaste, solamente tienes que ingresar la sigla con la cual ingresaste la consola y listo la consola fue eliminada
Sexto: Si ingresaste a la cuarta opcion puedes ver todas las consolas que tienes ingresadas, te van aparecer sus respectivos precios, stock, nombres, fabricantes
Septimo: Si ingresaste a la quinta opcion y ultima, puedes salir del programa y puedes cerrar todo
